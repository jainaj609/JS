Question 1: Find the number of digits?

Take a number from the user and print the count of digits in that number.
Note: You have to complete Find_Digits. No need to take any input.

Input Format

The input contains a single number N .

Output Format

Return the count of digits in the given number.

Example

Sample Input
1234
Sample Output
4

Ans.

                const Find_Digits = (N) => 
                {
                     let count=0;
                     while(N!=0){
                       count++;
                       N = Math.floor(N/10);
                     }
                     return count;
                };

Note: You have to complete findSum function. No need to take any input.

Input Format

The first and the only line of the input contains an integer N .

Output Format

Return the sum of all even integers from 1 upto N (N inclusive).

Example

Sample Input

Sample Output

Ans.

                const findSum = (n) => {
                    let it, ans=0;
                    for(it=1; it<=n; it++){
                       if(it%2===0){
                         ans+=it;
                       }
                     }
                     return ans;  
                };

Question 7: Check whether a Number is a prime or not?

Write a program which takes a number from user and check whether number is prime number or not a prime number.
Note: You have to complete Prime_Check. No need to take any input.

Input Format

The input contains a single number N .

Output Format

Return "YES" if the given number is Prime, else print "NO"(without quotes).

Example

Sample Input 1
7

Sample Output 1
YES

Sample Input 2
4

Sample Output 2
NO

Ans.

                    const Prime_Check = (N) => {
                        let s = "";
                        if (N == 1) {
                            s = "NO";
                        }
                        for (let i = 2; i < N; i++) {
                            if (N % i == 0) {
                                s = "NO";
                            }
                        }
                        if (s == "NO") {
                            return "NO";
                        } else {
                            return "YES";
                        }
                    };
